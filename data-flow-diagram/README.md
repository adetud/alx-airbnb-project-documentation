# Data Flow Diagram (DFD)

This diagram shows how data moves through the Airbnb Clone backend system.  
It covers the process of searching for properties, creating bookings, processing payments, and sending confirmations.

- **External Entities:** User
- **Processes:** Search Properties, Create Booking, Process Payment, Send Confirmation  
- **Data Stores:** Users DB, Properties DB, Bookings DB, Payments DB
