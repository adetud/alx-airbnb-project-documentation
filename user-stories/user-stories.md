# User Stories

1. As a user, I want to register and log in so that I can access my account securely.
2. As a host, I want to list my properties so that travelers can book them.
3. As a guest, I want to search and book properties so that I can find accommodation easily.
4. As a user, I want to make secure payments so that my transactions are safe.
5. As an admin, I want to manage users and properties so that I can maintain the platform.
